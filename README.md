## âœ¨ Features

- **Browse Icons**: View icons from multiple popular icon packs in one place.

- **One-Click Copy**: Click an icon to copy its class name to your clipboard.

- **Search**: Quickly filter icons by name or related keywords.

- **Configurable**: Choose which icon packs to display in the library.

## ğŸ“ Configuration

- `iconsLibrary.enabledPacks`: Control which icon packs are displayed.
