## ‚ú® Features

This extension provides a view of free icons from different icon packs. You can copy the icon class name to your clipboard and use it in your projects.

## üìö Packs

* [VS Code Codicons](https://microsoft.github.io/vscode-codicons/dist/codicon.html)
* [Font Awesome Free](https://fontawesome.com/)
* [PrimeIcons](https://primefaces.org/primeicons/)
* [Bootstrap Icons](https://icons.getbootstrap.com/)
* [Material Symbols](https://fonts.google.com/icons)

## üìù Configuration

You can configure which icon packs to display using the `iconsLibrary.enabledPacks` setting.
