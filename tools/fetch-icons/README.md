Criteria for fixing:
  No Tags: The icon has no tags or the list is empty.
  Too Few Tags: The icon has fewer than 5 tags.
  Low Quality Tags: The icon contains tags that are:
  Redundant: The tag is just a part of the icon name (e.g., tag "menu" for icon "menu-bar").
  Blacklisted: The tag is in the TAG_BLACKLIST (e.g., "icon", "vector", "outline").
